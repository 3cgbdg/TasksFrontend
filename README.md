# Tasks app

Full-stack Task application using:

- **Frontend:** Next.js
  
---

## Setup Instructions

### 1. Clone the repo

```bash
git clone <your-repo-url>
```


### 2. Frontend Setup (Next.js)
```bash
cd frontend
npm install
#Start frontend
npm run dev
```
### 3. Create .env
```bash
NEXT_PUBLIC_API_URL=http://localhost:5200
```
